# hexo-theme-pillars

Based on Gatsby theme [Lumen](https://lumen.netlify.com/).

### Install

```bash
$ git clone https://github.com/josevh/hexo-theme-clarity.git themes/clarity
```

### Enable

Modify the `theme` setting in `_config.yml` to `pillars`.

### Update

```bash
cd themes/clarity
git pull
```
